### Как установить и использовать(подробная инструкция для разворота из под Windows):

1. Определите разрядность операционной системы (Мой компьютер -> Свойства)

2. Скачайте установочные файлы, включающие среду Python 3.5.2, OpenCV 3.1 + contrib и NumPy
	2.1. Для Win32 скачайте установочные файлы отсюда: https://drive.google.com/drive/folders/0ByukWWVoeoC0ZEE1Y2J6VmV5SWM?usp=sharing
	2.2. Для Win64 скачайте установочные файлы отсюда: https://drive.google.com/drive/folders/0ByukWWVoeoC0dm5uQ1BXWG9NVlU?usp=sharing
	
3. Установите Python 3.5.2. на рабочую машину
	3.1. Для Win32 запустите python-3.5.2.exe
	3.2. Для Win64 запустите python-3.5.2-amd64.exe
	3.3. В открывшемся окне необходимо СНЯТЬ галочку Add Python 3.5 to PATH
	3.4. Нажать Customize Installation
	3.5. В открывшемся окне поставить все галочки
	3.6. В следующем окне изменить путь установки среды и НЕ МЕНЯТЬ галочки
		3.6.1. Для Win32 указать C:\Python35-32
		3.6.2. Для Win64 указать C:\Python35-64
	3.7. Нажать кнпоку Install и дождаться конца установки
	
4. Установить библиотеку NumPy
	4.1. Запустить консоль (Пуск -> набрать в строке поиска cmd -> Выбрать пункт Командная строка)
	4.2. Перейти в директорию <Корневая директория установленного питона>/Scripts. Например:
		4.2.1. Для Win32 выполнить команду cd C:\Python35-32\Scripts
		4.2.2. Для Win64 выполнить команду cd C:\Python35-64\Scripts
	4.3. Установить библиотеку NumPy из whl файла, скачанного на шаге 2.
		4.3.1. Для Win32 скопировать файл numpy-1.11.2+mkl-cp35-cp35m-win32.whl в директорию C:\Python35-32\Scripts.
		4.3.2. Для Win32 выполнить в консоли команду pip install numpy-1.11.2+mkl-cp35-cp35m-win32.whl и дождаться конца установки
		4.3.3. Для Win64 скопировать файл numpy-1.11.2+mkl-cp35-cp35m-win_amd64.whl в директорию C:\Python35-64\Scripts.
		4.3.4. Для Win64 выполнить в консоли команду pip install numpy-1.11.2+mkl-cp35-cp35m-win_amd64.whl и дождаться конца установки
	
5. Установить библиотеку OpenCV
	5.1. Запустить консоль (Пуск -> набрать в строке поиска cmd -> Выбрать пункт Командная строка)
	5.2. Перейти в директорию <Корневая директория установленного питона>/Scripts. Например:
		5.2.1. Для Win32 выполнить команду cd C:\Python35-32\Scripts
		5.2.2. Для Win64 выполнить команду cd C:\Python35-64\Scripts
	5.3. Установить библиотеку OpenCV из whl файла, скачанного на шаге 2.
		5.3.1. Для Win32 скопировать файл opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win32.whl в директорию C:\Python35-32\Scripts.
		5.3.2. Для Win32 выполнить в консоли команду pip opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win32.whl и дождаться конца установки
		5.3.3. Для Win64 скопировать файл opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win_amd64.whl в директорию C:\Python35-64\Scripts.
		5.3.4. Для Win64 выполнить в консоли команду pip install opencv_python-3.1.0+contrib_opencl-cp35-cp35m-win_amd64.whl и дождаться конца установки
		
6. Выкачать(склонировать) репозиторий https://github.com/DeadDreamer/adblockvr

7. Проверить работоспособность проекта
	7.1. Запустить консоль (Пуск -> набрать в строке поиска cmd -> Выбрать пункт Командная строка)
	7.2. Перейти в директорию с выкачанным проектом, посредством команды cd. Например: cd C:\dev\adblockvr
	7.3. Запустить проект в режиме захвата видео с камеры. Для этого выполнить команду: py -3.5 from_camera.py
	7.4. Запустить проект в режиме анализа видеофайла. Для этого выполнить команду: py -3.5 from_file.py
	7.5. Если в обоих случаях появилась форма, то, поздравляем, Вы все сделали верно

Удачи!:)
